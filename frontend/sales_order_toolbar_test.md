# 销售订单显示页面工具栏测试

## 新增功能

### 工具栏设计
- **位置**：页面顶部，标题下方
- **样式**：灰色背景，圆角边框，带阴影效果
- **布局**：左侧显示"操作工具栏"文字，右侧显示按钮组

### 按钮功能

#### 1. 查询按钮
- **图标**：搜索图标 (FiSearch)
- **颜色**：蓝色 (colorScheme="blue")
- **功能**：查询销售订单数据
- **当前实现**：控制台输出"查询销售订单"

#### 2. 新增按钮
- **图标**：加号图标 (FiPlus)
- **颜色**：绿色 (colorScheme="green")
- **功能**：新增销售订单
- **当前实现**：控制台输出"新增销售订单"

#### 3. 删除按钮
- **图标**：垃圾桶图标 (FiTrash2)
- **颜色**：红色 (colorScheme="red")
- **功能**：删除选中的销售订单
- **当前实现**：控制台输出"删除销售订单"
