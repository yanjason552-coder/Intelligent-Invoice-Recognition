# 销售订单显示页面图标工具栏测试

## 修改内容

### 页面结构变更
- **移除标题**：删除了"销售订单 - 显示"标题
- **工具栏位置**：工具栏现在是页面的第一个元素
- **布局**：按钮从左向右排列，左对齐
- **样式**：只显示图标，不显示文字
- **提示**：鼠标悬停时显示文字提示
- **尺寸**：统一按钮尺寸 40x40px

### 按钮功能

#### 1. 查询按钮
- **图标**：搜索图标 (FiSearch)
- **颜色**：蓝色 (colorScheme="blue")
- **提示文字**：查询
- **功能**：查询销售订单数据

#### 2. 新增按钮
- **图标**：加号图标 (FiPlus)
- **颜色**：绿色 (colorScheme="green")
- **提示文字**：新增
- **功能**：新增销售订单

#### 3. 删除按钮
- **图标**：垃圾桶图标 (FiTrash2)
- **颜色**：红色 (colorScheme="red")
- **提示文字**：删除
- **功能**：删除选中的销售订单

## 技术实现

### 组件结构
```tsx
<Box p={6}>
  {/* 工具栏 */}
  <Flex 
    bg="gray.50"           // 工具栏背景色
    p={3}                  // 内边距
    borderRadius="md"      // 圆角
    mb={6}                 // 下边距
    border="1px"           // 边框
    borderColor="gray.200" // 边框颜色
    justify="flex-start"   // 左对齐
    align="center"         // 垂直居中
    gap={2}                // 按钮间距
  >
    <Button
      colorScheme="blue"
      variant="outline"
      size="sm"
      onClick={handleQuery}
      title="查询"          // 悬停提示
      minW="40px"          // 最小宽度
      h="40px"             // 固定高度
    >
      <FiSearch />
    </Button>
    // ... 其他按钮
  </Flex>
  
  <Text>这是销售订单显示页面，用于查看销售订单的详细信息。</Text>
</Box>
```

### 关键特性
- **无标题设计**：页面顶部直接显示工具栏
- **title属性**：提供鼠标悬停提示
- **统一尺寸**：所有按钮都是40x40px
- **图标居中**：图标在按钮中居中显示
- **间距合理**：按钮之间有2px间距
