# 侧边栏宽度优化测试

## 修改内容

### 宽度调整
- **原宽度**：`minW="xs"` (约320px)
- **新宽度**：`w="213px"` (原宽度的2/3)
- **固定宽度**：使用`minW`和`maxW`确保宽度固定

### 技术实现
```tsx
<Box
  display={{ base: "none", md: "flex" }}
  position="sticky"
  bg="bg.subtle"
  top={0}
  w="213px"           // 固定宽度
  minW="213px"        // 最小宽度
  maxW="213px"        // 最大宽度
  h="calc(100vh - 22px)"
  p={4}
  overflowY="auto"
  overflowX="hidden"
>
```

### 布局优化效果

#### 1. 侧边栏区域
- **宽度减少**：从320px减少到213px
- **节省空间**：节省约107px的宽度
- **功能保持**：所有菜单功能保持不变

#### 2. 主工作区域
- **宽度增加**：主工作区域增加约107px
- **空间利用**：更多空间用于表格和内容显示
- **用户体验**：表格可以显示更多列，减少水平滚动

#### 3. 响应式设计
- **移动端**：不受影响，仍使用抽屉模式
- **桌面端**：固定宽度213px
- **兼容性**：保持原有的响应式行为
