## 使用方法
1. 复制整个模板文件集到您的项目目录
2. 将模板中的占位符替换为实际值：
   - `{业务模块}` → 实际的业务模块名（如：product_category）
   - `{实体名}` → 实际的实体名（如：ProductCategory）
   - `{字段1}` → 实际的字段名（如：category_code）
3. 按照开发流程说明逐步实施

## 命名规则
- 数据库表名：snake_case
- 类名：PascalCase
- 文件名：camelCase
- API路径：kebab-case

## 技术栈
- 后端：FastAPI + SQLModel + PostgreSQL
- 前端：React + TypeScript + Chakra UI
- 数据库：PostgreSQL
