# 侧边栏滚动功能测试

## 修改内容

### 1. 桌面版本滚动
- 为桌面侧边栏添加了 `overflowY="auto"` 属性
- 添加了 `overflowX="hidden"` 防止水平滚动
- 当菜单内容超过视口高度时，会出现垂直滚动条

### 2. 移动端滚动
- 为移动端抽屉的 `DrawerBody` 添加了 `overflowY="auto"` 属性
- 确保移动端菜单也能正常滚动

## 测试步骤

### 桌面端测试
1. 在桌面浏览器中打开应用
2. 展开左侧菜单的所有分组
3. 验证当菜单内容超过屏幕高度时：
   - 右侧出现滚动条
   - 可以上下滚动查看所有菜单项
   - 滚动条样式符合系统主题

### 移动端测试
1. 在移动设备或浏览器开发者工具中切换到移动视图
2. 点击汉堡菜单按钮打开侧边栏
3. 展开所有菜单分组
4. 验证抽屉内容可以正常滚动

## 预期结果

- ✅ 桌面端菜单支持垂直滚动
- ✅ 移动端抽屉支持垂直滚动
- ✅ 滚动条样式美观
- ✅ 菜单项点击功能正常
- ✅ 展开/收起功能正常

## 技术实现

```tsx
// 桌面版本
<Box
  overflowY="auto"      // 垂直滚动
  overflowX="hidden"    // 隐藏水平滚动
  h="calc(100vh - 22px)" // 固定高度
>

// 移动端版本
<DrawerBody overflowY="auto">
  <Flex flexDir="column" justify="space-between" minH="100%">
```
